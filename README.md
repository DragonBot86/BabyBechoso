### PROCESO DE INSTALACION
Insttala archivos necesarios
```bash
- pkg upgrade && update
- pkg install nodejs
- pkg install git
- pkg install bash
```

 ```bash
> git clone https://github.com/thechoute/BabyBechoso
```

```bash
- cd BabyBechoso
- bash install.sh
```
PARA INICIAR EL BOT

 ```bash
- npm start
```
